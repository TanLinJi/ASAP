1. 项目目标与核心 claim

方法名称：ASAP = Adaptive Spherical Anomaly-Guided Purification（自适应球形异常引导净化）。

论文目标：投稿 IEEE ICASSP 2026，提出一个面向 LiDAR 3D 目标检测的检测器无关（detector-agnostic）、仅推理时使用（inference-only）的对抗净化框架。

摘要里要支撑的 4 个核心 claim：
  1. ASAP 不重训任何检测器，纯推理时净化。
  2. ASAP 在多种检测器架构上都有效（detector-agnostic）。
  3. ASAP 在多个数据集（KITTI 与 Waymo）上都有效。
  4. ASAP 把对抗攻击的 Attack Success Rate 降低 60%，同时干净场景检测性能基本不下降。

2. 缩写一览（中英对照）

缩写英文全称中文含义 / 在 ASAP 中的作用
ASAPAdaptive Spherical Anomaly-Guided Purification本论文方法名。
SPUSpherical Purification Unit球形净化单元。点云被划分成的局部球。
KNNk-Nearest Neighborsk 近邻。M1 用 k 近邻距离决定球半径。
PCAPrincipal Component Analysis主成分分析。M2 用它估计局部各向异性。
vMFvon Mises-Fisher distribution球面上的 “高斯”。M2 用它估计角度集中度 $\kappa$。
VP-SDEVariance-Preserving Stochastic Differential Equation方差保持随机微分方程。一种连续扩散过程，M3 用它做点云级扩散净化。
ROCReceiver Operating Characteristic受试者工作特征曲线，用于阈值 $\tau$ 的标定。
Youden-JYouden Index $J$$J = \text{TPR} - \text{FPR}$，从 ROC 曲线选最优阈值。
AP / mAP(mean) Average Precision检测平均精度。KITTI 用类别 AP，Waymo 用 mAP。
mAPHmean Average Precision weighted by HeadingWaymo 评测中考虑朝向的 mAP。
NDSnuScenes Detection ScorenuScenes 综合得分（暂留作未来扩展）。
ASRAttack Success Rate攻击成功率，衡量攻击让检测器漏检/错检的比例。
SOR / RORStatistical / Radius Outlier Removal两种经典点云去噪基线。

3. 方法模块代号 M1 / M2 / M3

论文 Method 章节（03_method.md）将方法切分为三个模块。所有代码、消融、推导都按这套代号引用。

代号 模块名 核心数学对象 代码落点
M1 Adaptive Radius
自适应半径 SPU 划分		 外半径 $r_1(p) = \mathrm{clip}(\alpha\, d_k(p), [r_{\min}, r_{\max}])$，
内半径 $r_2(p) = \beta\, r_1(p)$，默认 $\beta = 0.67$。
默认 $\alpha = 2,\ k = 16,\ r_{\min} = 0.10\text{ m},\ r_{\max} = 0.40\text{ m}$。 		src/asap/geometry/
M2 Anomaly Scorer
几何异常评分		 4 个标量特征 $\{f_c, f_a, f_v, f_d\}$ 融合得到 $s(p) \in [0, 1]$；
阈值 $\tau$ 由 ROC 上的 Youden-$J$ 标定。 		src/asap/scoring/
M3 Selective Diffusion
选择性扩散净化		 若 $s(p) > \tau$，对内点集 $P_{\mathrm{inner}}(p)$ 做 VP-SDE 短轨迹正向加噪 + 反向去噪；
其他区域保持不变。 		src/asap/diffusion/, src/asap/pipeline/
注：每个模块在实验代号体系中都有对应的 消融实验（见 E5）。改动模块时，请同步更新对应消融。

关键约束：

  • E1 必须早于对应数据集和检测器的 E4。
  • E2 攻击必须先生成受扰 val 集，才能跑 E3 / E4。
  • E5 消融建议在 KITTI + PointPillars 上完成（最便宜）。
  • E6 成本测量建议与 E4.1 同时进行（同一台机器，避免硬件差异）。
